# NoteApp

ブラウザで動く，手書き中心のシンプルなノートアプリです。

## 現在の状態

HTML / CSS / JavaScriptだけで動く最小版です。

- ノートの新規作成
- タイトル編集（一覧・編集画面）
- キャンバスへの手書き
- ノート一覧でのタグアイコン編集とタグ絞り込み
- 固定線幅のペン
- 消しゴム選択中の通常/ストローク単位切り替え
- 保存されないレーザーポインター
- Pencilモードとタッチ描画モードの切り替え
- Pencilモード中の二本指ダブルタップでundo
- ペン選択中の黒，青，赤の色選択
- Lucideベースの線画アイコン
- 一番細い線幅に固定
- A4，大学ノート風罫線固定の紙面
- 紙面の拡大縮小
- 最後のページに描画すると空白ページを自動追加
- 末尾の描画が消えると余分な空白ページを自動削除
- 現在ページの上への空白ページ追加
- 選択ページの削除
- undo / redo
- ノート一覧からのノート削除
- PDFエクスポート
- タイトル検索
- 最終更新時刻表示
- IndexedDBへの自動保存
- PWAインストール対応
- ノート一覧からのPWAインストール
- Service Workerによる基本ファイルのオフラインキャッシュ
- モバイルファーストのレスポンシブレイアウト
- ノート一覧ページと手書き編集ページの分離
- 編集中はノート一覧を隠す集中モード

まだインポートは未実装です。

## 動かし方

`index.html` をブラウザで開きます。

PWA機能を確認する場合は，開発用サーバーで開きます:

```sh
python3 -m http.server 5173
```

その後，ブラウザで `http://localhost:5173` を開きます。

`file://` で直接開いた場合，Service Workerとインストール機能は有効になりません。

ノート一覧のインストールボタンからPWAをインストールできます。ブラウザが自動インストール画面に対応していない場合は，手動追加の案内を表示します。
