# NoteApp

ブラウザで動く，手書き中心のシンプルなノートアプリです。

## 現在の状態

HTML / CSS / JavaScriptだけで動く最小版です。

- ノートの新規作成
- タイトル編集
- キャンバスへの手書き
- ペン
- 消しゴム
- 黒，青，赤の色選択
- 丸の大きさで選ぶ線幅選択
- A4固定の紙面
- 無地，罫線，方眼の背景変更
- 紙面の拡大縮小
- ページ追加，削除，前後移動
- undo / redo
- ページクリア
- ノート一覧からのノート削除
- タイトル検索
- 最終更新時刻表示
- `localStorage` への自動保存
- モバイルファーストのレスポンシブレイアウト
- ノート一覧ページと手書き編集ページの分離
- 編集中はノート一覧を隠す集中モード

まだPWA化，Service Worker，IndexedDB，エクスポート/インポートは未実装です。

## 動かし方

`index.html` をブラウザで開きます。

開発用サーバーで確認する場合:

```sh
python3 -m http.server 5173
```

その後，ブラウザで `http://localhost:5173` を開きます。
